# Nanobot 任務管理網站

一個簡單的個人任務管理前端應用程式，可協助追蹤和管理日常任務。

## 功能特點

- ✅ 新增任務
- ✅ 標記任務完成狀態
- ✅ 刪除任務
- ✅ 任務統計（總數、已完成、進行中）
- ✅ 資料本地持久化（localStorage）
- ✅ 響應式設計
- ✅ 現代化 UI

## 使用方式

1. 在輸入框中輸入任務內容
2. 點擊「新增任務」或按 Enter 鍵添加
3. 點擊checkbox來標記任務完成/未完成
4. 點擊「刪除」按鈕移除任務

## 技術棧

- HTML5
- CSS3（含響應式設計）
- 原生 JavaScript
- LocalStorage（資料持久化）
